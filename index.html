Warcraft Forge

Warcraft Forge utilizes Blizzard's World of Warcraft API to allow users to lookup their character or others. It provides details on their stats, talent choices and artifact traits.

It was built with Facebook's React, ES6 and is hosted on Heroku. An ExpressJS server is used to create an API endpoint to make the request to Blizzard's API and sends it back to the client.

Meteorite Map

NASA's data portal provides tons of different data sets. I came across one I thought was interesting which listed meteorite landings. I grabbed .csv copy of that data. Then I grabbed a .shp file of all the countries of the world and converted that to GeoJSON. With D3.js I rendered a SVG of the world map and plotted all of the meteorites that landed between 2000 and 2016.

Weather

Before I got into React I was using Angular to help with the development of a web application for a company. It features a search box where a user can type in their City and State to get a short weather forecast for their area.

This website also features a small Express backend to grab the city and state name use Google's geolocation API to convert it to longitude/latitude. It then sends a request to ForecastIO's API and returns the response back to the front-end.

JS Starter

With the emergence of Webpack I began feeling left behind since I was using the same gulpfile I made months ago when I first starting learning Web Development. I loved how create-react-app provided a way for me to quickly get started on a new idea. I wanted to create something that was framework or library agnostic so I did not have to feel bogged down by all the choices out there in the JavaScript ecosystem.

This features a Mock API, development sharing with localtunnel, CI through Travis CI, JS linting, testing, and a production build pipeline.
